# Smooth Air Writing (Camera Sensor)

A browser-based hand-tracking writing project using your camera.

## Features
- Smooth air writing with hand tracking
- No spark particle effect
- Stable pinch-to-draw control with hysteresis
- Adjustable smoothness, line width, and pinch thresholds
- Undo, clear, and save as PNG

## Gesture Controls
- **Draw:** pinch thumb + index finger
- **Pause:** open palm or release pinch

## Run
Open [index.html](index.html) in a modern browser (Chrome/Edge) and allow camera permission.

> If camera access is blocked, run with a local server and try again.
